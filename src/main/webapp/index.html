What's SalaryMaster about?

SalaryMaster provide the lastest official published H1B salary data from government. We collect wage data from almost all of the companies in America. SalaryMaster makes it searchable, filterable and provides statistic wage data for each recored company. It also enable you to search by geographic data.

Where is the source data?

All the data resource are from data.gov.
